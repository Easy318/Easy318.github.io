常用数据类型:

python类型 映射到数据库的对应类型 说明
Integer int 整型
Float float 浮点类型(占32位)
Double double 双精度浮点类型(占64位)
String varchar 可变字符类型
Boolean tinyint 布尔类型
DECIMAL decimal 定点类型
Enum enum 枚举类型
Date datetime 时间类型，只能存储年月日
DateTime datetime 时间类型，可以存储年月日时分秒毫秒等
Time datetime 时间类型，只能存储时分秒
Text text 存储长字符串(大概6w多字符)
LONGTEXT longtext 长文本类型

Column常用参数:

  • primary_key：设置字段为主键
  • autoincrement：设置字段为自增长的
  • default：设置字段的默认值,常用于发表时间这些字段上
  • nullable：指定某个字段是否可以为空,默认值是True
  • unique：指定某个字段的值是否唯一,默认是False
  • onupdate：在数据更新的时候会调用这个参数指定的值或者函数
    • 注意
    • 在第一次插入这条数据的时候，不会用onupdate的值，只会使用default的值
    • 常用于每次更新数据的时候都要更新字段,比如update_time
  • name：指定ORM模型中某个属性映射到表中的字段名
    • 注意
    • 如果不指定，那么会使用这个属性的名字来作为字段名
    • 如果指定了，就会使用指定的这个值作为参数
